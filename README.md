# LostFound Campus

LostFound Campus is a web-based Lost & Found portal designed for college students to report, search, and recover lost items easily.

Project Description

This application allows users to:

    Report lost items
    Post found items
    Search for items
    Claim items securely
    Track item status

It helps students reconnect with their lost belongings efficiently.

##  Features

    User Registration & Login
    Post Lost Items
    Post Found Items
    Search Functionality
    Claim Request System
    User Dashboard
    Image Upload
    Status Tracking (Open → Closed)


## Technologies Used

   Frontend: HTML, CSS, JavaScript
   Backend: Python (Flask)
   Database: MongoDB

##  How to Run the Project

1. Clone the repository:
   git clone https://github.com/yourusername/lostfound-campus.git

2. Go to the project folder:
   cd lostfound-campus

3. Create virtual environment:
   python -m venv venv

4. Activate environment:
   venv\Scripts\activate

5. Install dependencies:
   pip install -r requirements.txt

6. Create `.env` file and add:
   SECRET_KEY=your_secret_key
   MONGO_URI=your_mongodb_uri

7. Run the app:
   python app.py

8. Open browser:
   http://127.0.0.1:5000
